ORDENAR UNA LISTA POR APELLIDOS

Objetivo: Introducir un listado de nombre con apellido y buscar la lógica para ordenarlos todos. Previamente se mostrarán los conceptos necesarios.

El método .sort()

Es un método incorporado en JavaScript que se utiliza para ordenar los elementos de un array. Por defecto, ordena los elementos en función de su representación en formato de cadena (orden lexicográfico), lo que significa que los números y las letras se ordenan alfabéticamente. Sin embargo, también se puede personalizar la función de comparación para ordenar los elementos de acuerdo con un criterio específico.

LocaleCompare()

Es un método en JavaScript que se utiliza para comparar dos cadenas de texto y determinar su orden lexicográfico (alfabético) según las reglas de ordenación del idioma y localización específicos. A continuación hipervínculo a la documentación: Documentación del método.
